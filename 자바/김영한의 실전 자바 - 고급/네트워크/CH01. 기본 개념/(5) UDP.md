# UDP (User Datagram Protocol)

- UDP는 TCP와 같은 레이어에 존재한다.
- 즉, TCP와 UDP 둘 중 하나를 선택해야 한다.
- 특징
  - 연결 지향적이지 않다.
  - 데이터 전달 보증이 없다.
  - 순서도 보장되지 않는다.
  - **빠르다.(중요)**
- IP와 별로 다른 것이 없으며 IP에 추가되는 것은 다음과 같다.
  - **PORT** (전송 계층이기 때문에 필요, 하나의 IP에 여러 애플리케이션이 실행될 수 있도록 함)
  - **체크섬**
- 때문에 애플리케이션 수준에서 추가적인 작업이 필요하다.
