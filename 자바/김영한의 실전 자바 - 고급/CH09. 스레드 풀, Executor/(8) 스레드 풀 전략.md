# 스레드 풀 전략

- `corePoolSize`, `maximumPoolSize` 등을 조합하여 나에게 맞는 스레드 풀 전략을 선택적으로 사용할 수 있다.
- 자바는 3가지의 대표적인 기본 전략을 제공한다.

## 단일 스레드 풀 전략

```java
new ThreadPoolExecutor(1, 1, 0L, TimeUnit.MILLISECONDS, new LinkedBlockingQueue<Runnable>());
```

- 스레드를 1개만 사용한다.
- 블로킹큐의 용량 제한이 없다.
- 간단한 테스트 용도 정도로 사용한다.
